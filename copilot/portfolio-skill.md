# Portfolio Design Rules

Create a modern developer portfolio with a dark theme.

## Colors
- Primary background: #0A0F1F
- Secondary background: #111827
- Primary accent: #2563EB
- Text color: #E5E7EB

## Style
- Minimalist
- Professional
- Similar to modern SaaS landing pages
- Inspired by Platzi blog layouts
- Rounded cards
- Smooth animations
- Responsive design
- Use gradients in blue tones

## Sections
- Hero
- About Me
- Experience
- Education
- Skills
- Projects
- Contact

## Typography
- Large bold headings
- Clean spacing
- Modern sans-serif fonts

## Technologies
Use:
- Angular
- TailwindCSS
- TypeScript

## Cards
Cards should have:
- Rounded borders
- Shadow
- Dark backgrounds
- Hover effects

## Skills Section
Display technologies as badges:
- Angular
- Laravel
- Flutter
- PHP
- MySQL
- Docker
- Git

## Experience Section
Use a vertical timeline design.

## Projects
Projects should appear in modern cards with:
- Title
- Description
- Technologies
- GitHub button
- Demo button
