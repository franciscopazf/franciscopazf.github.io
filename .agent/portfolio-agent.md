# Portfolio Agent Instructions

You are building a modern developer portfolio using Astro.

The design must look professional, premium and modern,
inspired by modern SaaS landing pages and Platzi layouts.

# Stack

Use:

- Astro
- TailwindCSS
- TypeScript

# Theme

Create a dark UI with:

- Black backgrounds
- Dark blue gradients
- Glassmorphism cards
- Smooth hover animations

Main colors:

- #0A0F1F
- #111827
- #2563EB
- #3B82F6

# Architecture

The project MUST be componentized.

Create reusable UI components.

Use this structure:

src/
  components/
    ui/
    layout/
    sections/

# Required reusable components

Create:

- Button.astro
- Card.astro
- Badge.astro
- SectionTitle.astro
- Container.astro

# Required sections

Create:

- Hero
- About
- Experience
- Education
- Skills
- Projects
- Contact

# Fonts

Use Google Fonts:
- Inter

Fonts must be reusable globally.

# Styling Rules

- Use rounded-2xl cards
- Use spacing generously
- Use subtle borders
- Use hover animations
- Use transitions
- Responsive design required
- Mobile-first

# Skills Section

Show skills as modern badges.

Include:

- Angular
- Laravel
- Flutter
- PHP
- Docker
- MySQL
- Git
- TypeScript

# Experience Section

Create a vertical timeline design.

# Projects Section

Projects must include:

- Title
- Description
- Technologies
- GitHub button
- Demo button

# Data

Do not hardcode data directly in sections.

Create reusable files inside:

src/data/

# Performance

Keep components lightweight.

# Design Goal

The final design should look like a modern developer
resume + portfolio website.
